**API de compras y Productos **
Este proyecto es una **API** checha con Node.js y MySQL que proporciona endpoints para la gestion de un CRUD
que utiliza logicas de un negocio, como validaciones de inventario y transacciones de base de datos.
Utilizado: **Node.js**, **Express**, **Mysql**, **dotenv** y **AWS** (alojamiento de la base de datos MYSQL)
